# React HelloWorld
## Node.js
Install Node.js to compile & run TypeScript.  

https://nodejs.org/en/  
 note : The stable, even version is recommended.  

Version Check  
$ node -v  

## Code
```TypeScript
function App() {
  return (
    <div className="App">
      <header className="App-header">
        <img src={logo} className="App-logo" alt="logo" />
        <p>
          Hello World !
        </p>
      </header>
    </div>
  );
}
```

## OutputSample
$ npm run start
